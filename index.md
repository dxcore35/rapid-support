## FAQ

### I bought Rapid but cannot find purchase restore

Rapid is a one-time purchase with no subscription. If you changed devices, make sure you are signed in with the same Apple ID used for purchase.

### Which formats are supported in v1.0?

Rapid v1.0 supports PDF import, web links (URL), plain text, and clipboard text.
